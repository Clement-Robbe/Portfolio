Site pour un tournoi de jeu vidéo

Il s'agit d'un site pour un tournoi de jeu vidéo en ligne. Il est réalisé avec PHP en architecture MVC avec Symfony. Il n'est biensûr plus maintenu depuis la fin du tournoi.

C'est ce projet qui m'a permis de mettre en pratique mes compétences en développement web et en gestion de projet pour la première fois.

les compétences mises en oeuvre sont :

  • Gestion de projet
  • Gestion de base de données
  • Administration de serveur
  • les langages : Html, CSS, JavaScript, PHP, MySQL
  • les frameworks : Symfony, Bootstrap

Il a été réalisé à l'occasion de la 4ème édition du tournoi e-sport "Game_osu" en 2019 en partenariat avec Logitech et sous la tutelle d'un influenceur français connu.

quelques chiffres du tournoi :

    Diffusion française sur Twitch
  • 36.000 spectateurs uniques
  • 10.000 vues simultanées
    Diffusion internationale sur Twitch
  • 6.000 spectateurs uniques
  • 2.500 vues simultanées
